# RoboRacers v2.0

A fast-paced introduction to robotics. Write programs to get the robot to move and use a sensor to let the robot see and follow a line.

## Objectives

In this challenge you will:

- Learn how to create and run programs for the robot.
- Master moving the robot around.
- Learn how robots can "see" the world with a reflectance sensor.
- Race along the racetrack and tune the robot so it completes it as fast as possible.

## Instructions

Follow the `RoboRacers.pdf` worksheet, using the programs in this release.

## Session Details

- **Name**: RoboRacers
- **Version**: v2.0
- **Duration**: 45-60 minutes
- **Suitable for**: Secondary school ages with no programming experience. Younger students may need a longer session.
